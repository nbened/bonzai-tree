# Bonzai Solidify Workflow

## Description
After completing any code generation or modification task, ALWAYS proactively offer to solidify the code using bonzai-burn. This removes technical debt and ensures production-ready code.

## When to Proactively Suggest Solidification
- After creating new files
- After modifying existing code
- After implementing a feature
- After ANY substantial code changes

Always ask: "Should I solidify this code?"
